# Technical Guide: Decoding SLY Composite Operations with Carbon Rust Decoders

This guide provides technical instructions for integrating the official Star Atlas Rust decoders (Carbon) into the `sae-main` project to analyze experimental composite transactions generated by the SLY Assistant script.

## 1. Context: SLY Composite Operations
The SLY script (experimental features) bundles multiple SAGE instructions into a single Solana transaction to optimize gas and execution time.
- **Key Features**: `queueExitWarpSubwarp`, `transportLoadUnloadSingleTx`, `minerSupplySingleTx`.
- **Challenge**: Standard Anchor decoders may fail on complex "remaining data" segments. Carbon decoders (Rust) provide superior accuracy via custom patches.

## 2. Rust Decoder Implementation
Modify the Rust binary in `sae-main/rust_decoder` to handle instruction arrays.

### File: `sae-main/rust_decoder/src/main.rs`
Add a dedicated mode for composite decoding:

```rust
use carbon_sage_starbased_decoder::SageDecoder;
use carbon_core::instruction::InstructionDecoder;
use solana_instruction::Instruction;
use solana_pubkey::Pubkey;
use std::str::FromStr;

/// Decodes an array of SAGE instructions passed as a JSON string
fn decode_composite_instructions(instructions_json: &str) -> serde_json::Value {
    let v: serde_json::Value = serde_json::from_str(instructions_json).unwrap_or(json!([]));
    let mut decoded_results = Vec::new();
    let decoder = SageDecoder;

    if let Some(ix_list) = v.as_array() {
        for ix_val in ix_list {
            let program_id_str = ix_val["programId"].as_str().unwrap_or("");
            let data_hex = ix_val["data"].as_str().unwrap_or("");
            
            if let (Ok(program_id), Some(data)) = (Pubkey::from_str(program_id_str), hex_to_bytes(data_hex)) {
                let solana_ix = Instruction {
                    program_id,
                    accounts: vec![], // Carbon decoders primarily use data discriminators
                    data,
                };

                if let Some(decoded) = decoder.decode_instruction(&solana_ix) {
                    decoded_results.push(json!({
                        "name": format!("{:?}", decoded.data),
                        "data": decoded.data,
                        "success": true
                    }));
                } else {
                    decoded_results.push(json!({"error": "Unknown instruction", "success": false}));
                }
            }
        }
    }
    json!(decoded_results)
}
```

## 3. TypeScript Integration
Update the backend to bridge the gap between the Node.js environment and the Rust binary.

### File: `sae-main/src/decoders/rust-wrapper.ts`
Add the composite decoding function:

```typescript
/**
 * Calls the Rust Carbon decoder for a batch of instructions.
 * @param instructions Array of { programId: string, data: string (hex) }
 */
export function decodeCompositeWithRust(instructions: any[]): any[] | null {
  try {
    const binPath = process.env.RUST_DECODER_BIN || './bin/carbon_decoder';
    const payload = JSON.stringify(instructions);
    
    const res = spawnSync(binPath, ['--mode', 'composite', payload], { 
      encoding: 'utf8', 
      maxBuffer: 10 * 1024 * 1024 
    });
    
    return res.stdout ? JSON.parse(res.stdout) : null;
  } catch (e) {
    return null;
  }
}
```

### File: `sae-main/src/decoders/composite-decoder.ts`
Integrate the Rust wrapper into the main decoding flow:

```typescript
import { decodeCompositeWithRust } from './rust-wrapper.js';

const SAGE_PROGRAM_ID = 'SAGE2HAwep459SNq61LHvjxPk4pLPEJLoMETef7f7EE';

export function decodeCompositeInstructions(txJson: any) {
  const instructions = txJson.data?.transaction?.message?.instructions || [];
  const sageIxs = instructions.filter((ix: any) => ix.programId === SAGE_PROGRAM_ID);
  
  if (sageIxs.length > 1) {
    const rustDecoded = decodeCompositeWithRust(sageIxs);
    if (rustDecoded) {
      return {
        isComposite: true,
        engine: 'carbon-rust',
        instructions: rustDecoded
      };
    }
  }
  return { isComposite: false, instructions: [] };
}
```
